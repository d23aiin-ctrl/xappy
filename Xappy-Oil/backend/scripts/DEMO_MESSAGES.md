## CANCEL/DISCARD

To cancel a report in progress:
```
cancel
```
```
no
```
```
discard
```

To start over:
```
start new report
```
Or just click "New chat" button

---

## TIPS

1. **Natural language works best** - Describe what happened naturally, the AI will extract the fields
2. **Include key details** - Date/time, location, what happened, type of hazard
3. **Use specific terms** - "slip trip fall", "equipment malfunction", "first aid" help with categorization
4. **Review before submit** - Check the confirmation card to verify all details are correct
5. **Click to edit** - If something is wrong, click the field in the draft card to fix it
